# Real-Time Fatigue and Posture Detection using Computer Vision

This project is a real-time health monitoring system built using Computer Vision and Machine Learning techniques.  
It detects eye fatigue (blink rate) and body posture using webcam input.

The system is designed for assistive AI applications such as smart glasses and wearable health monitoring systems.

---

## Features

- Real-time webcam monitoring
- Face detection
- Eye blink detection
- Posture detection
- Fatigue alert system
- Data logging to CSV
- Lightweight and suitable for edge devices

---

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Pandas
- Matplotlib
